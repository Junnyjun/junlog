# 1장. Java를 이해하기

## 1장. Java를 이해하기

Java는 현재 가장 널리 사용되는 소프트웨어 개발 기술 중 하나입니다. Java 기술은 단순한 프로그래밍 언어를 넘어, 애플리케이션을 개발하고 실행하기 위한 하나의 완전한 기술 체계를 구성하고 있습니다.

Java 기술 체계는 프로그램 실행을 담당하는 가상 머신(Java Virtual Machine), 다양한 개발 환경을 지원하는 표준 라이브러리, Java 프로그래밍 언어, 그리고 수많은 오픈소스 프레임워크와 도구들로 이루어져 있습니다.

오늘날 Java는 웹 서비스, 엔터프라이즈 시스템, 금융 플랫폼, 모바일 환경, 임베디드 장치 등 다양한 분야에서 활용되고 있으며, 오랜 기간 동안 소프트웨어 산업의 핵심 기술 중 하나로 자리 잡아 왔습니다.

Java가 널리 사용되는 이유는 단순히 언어 자체의 특징 때문만은 아닙니다. Java는 서로 다른 운영체제와 하드웨어 환경 위에서도 동일한 프로그램을 실행할 수 있는 플랫폼 독립성을 제공하며, 자동 메모리 관리와 풍부한 라이브러리를 통해 개발 생산성을 높여 왔습니다.

특히 Java 기술의 중심에는 JVM이 존재합니다. JVM은 하드웨어와 운영체제의 차이를 추상화하여 개발자가 특정 플랫폼에 종속되지 않고 애플리케이션을 개발할 수 있도록 지원합니다.

대부분의 개발자는 Java 언어와 라이브러리를 사용하여 애플리케이션을 개발하지만, 실제로 프로그램이 어떤 과정을 거쳐 실행되는지 깊이 이해할 기회는 많지 않습니다. 그러나 시스템 규모가 커지고 성능과 안정성이 중요해질수록 JVM 내부 동작 원리에 대한 이해는 점점 더 중요해집니다.

이 장에서는 Java 기술 체계가 어떻게 구성되어 있는지 살펴보고, Java가 걸어온 발전 과정과 다양한 가상 머신의 계보를 알아봅니다. 또한 Java 기술이 앞으로 어떤 방향으로 발전할 것인지 살펴보고, OpenJDK를 직접 빌드하는 과정을 통해 Java 플랫폼을 보다 깊이 이해해 보겠습니다.
